# pwa-ionic-timer
pwa-ionic-timer



```
git clone https://github.com/mrkamlesh/pwa-ionic-timer.git pwaTimer

npm i

ionic cordova plugin add cordova-plugin-insomnia
npm install @ionic-native/insomnia


ionic cordova plugin add cordova-plugin-navigationbar
npm install @ionic-native/navigation-bar


ionic cordova plugin add cordova-wheel-selector-plugin
npm install --save @ionic-native/wheel-selector

npm i ng-circle-progress

npm i angular-progress-bar --save


ionic serve


ionic cordova prepare
ionic cordova build android

ng add @angular/pwa --project app
ionic build --prod


cd www

npm install -g http-server@0.9.0
http-server -o


npm i -g firebase-tools

firebase logout/login

firebase init

firebase deploy


URL: https://pwatimer.firebaseapp.com/

```
